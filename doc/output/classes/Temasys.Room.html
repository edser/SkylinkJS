Parameters:

  • options JSON

    The options.

    • appKey String

      The App Key ID to connect to the App space to.

    • [name] String optional

      The Room name to connect to. The default is the App Key ID value.

    • [autoInit=true] Boolean optional

      The flag if init should be automatically called the moment the Room class object is constructed.

    • [requireWebRTC=true] Boolean optional

      The flag if WebRTC functionalities are required mandatorily. If the browser or device does not support WebRTC functionalities, a normal Signaling server connection commences.
