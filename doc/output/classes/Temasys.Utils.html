.createEventManager.once (event, callback, [condition], [fireAlways]) →   None

Function to subscribe to an event once.

Parameters

  • event - String

    The event to subscribe to once.

  • callback - Function

    The callback listener function.

  • condition - Function optional

    The condition function that is called when event is triggered. If condition is met (when function returns true), the callback listener function is triggered. The defaults is function () { return true; }.

  • fireAlways - Boolean optional

    The flag if callback listener function should always be triggered regardless as long as condition function is met. The defaults is false.

Returns

None
