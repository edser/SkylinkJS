Referencing Temasys Web SDK is available in Temasys CDN. The /0.7.x/ URI folder references the latest /0.7.x/ SDK versions, and /latest/ references the latest SDK release version. You may replace the /0.7.x/ in the examples below (e.g. /0.7.1/) to reference the specific version of the SDK you would like.

Uncompressed files are best for debugging issues and Compressed files are useful for production by reducing bandwidth downloads.

Bundled (Complete) files

This bundled files contains the required socket.io-client and Temasys AdapterJS dependencies together with the SDK.

Compressed:

<script src="https://cdn.temasys.io/skylink/skylinkjs/0.7.x/skylink.complete.min.js" type="text/javascript"></script>

Uncompressed:

<script src="https://cdn.temasys.io/skylink/skylinkjs/0.7.x/skylink.complete.js" type="text/javascript"></script>

SDK only files

This is useful for cases where you have to reference the other required socket.io-client and Temasys AdapterJS manually in the instance for customized AdapterJS dependency for custom Temasys WebRTC plugin or Screensharing extensions.

Compressed:

<script src="https://cdn.temasys.io/skylink/skylinkjs/0.7.x/skylink.min.js" type="text/javascript"></script>

Uncompressed:

<script src="https://cdn.temasys.io/skylink/skylinkjs/0.7.x/skylink.debug.js" type="text/javascript"></script>
